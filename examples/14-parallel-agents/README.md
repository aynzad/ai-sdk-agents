# 14 - Parallel Agents

Runs multiple agents in parallel with `Promise.all`, each analyzing the same topic from a different perspective, then combines results with a synthesizer agent.

## Key Concepts

- Running multiple `Runner.run()` calls in parallel with `Promise.all`
- Combining outputs from multiple agents into a single synthesis
- Different agent personas (optimist, pessimist, realist)

## Run

```bash
pnpm start
```

## Test

```bash
pnpm test
```
