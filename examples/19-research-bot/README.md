# 19 - Research Bot

Multi-agent research workflow demonstrating a planner → searcher → writer pipeline.

- **Planner** agent analyzes a topic and produces targeted search terms
- **Searcher** agents run in parallel, producing summaries and key facts for each term
- **Writer** agent synthesizes all research into a coherent report

## Run

```bash
pnpm install
pnpm start
```

## Test

```bash
pnpm test
```
