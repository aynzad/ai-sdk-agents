# 13 - Keyword Guardrail

Built-in guardrail helpers: `keywordGuardrail`, `maxLengthGuardrail`, and `regexGuardrail`. Demonstrates input guardrails that block dangerous keywords and output guardrails that enforce length limits and detect credit card patterns.

## Run

```bash
pnpm install
pnpm start
```

## Test

```bash
pnpm test
```
