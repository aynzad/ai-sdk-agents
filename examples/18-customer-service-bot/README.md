# 18 - Customer Service Bot

Multi-agent airline customer service bot with triage, FAQ, and booking agents.

The **Triage Agent** classifies user intent and routes to the appropriate specialist:

- **FAQ Agent** — answers common questions about baggage, check-in, and policies
- **Booking Agent** — handles seat information and seat changes

## Run

```bash
pnpm start
```

## Test

```bash
pnpm test
```
