# 07 - Agent Handoff

Demonstrates `handoff()` where a triage agent hands off the conversation to a Spanish-speaking agent when the user writes in Spanish.

## Key Concepts

- `handoff()` to declare agent handoff targets
- `Agent` with `handoffs` array
- `Runner.run()` with automatic handoff execution
- `RunResult.agent` to identify which agent responded
- `RunResult.steps` to inspect handoff events

## Run

```bash
pnpm start
```

## Test

```bash
pnpm test
```
